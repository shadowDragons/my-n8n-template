# 602在请求中自动管理用户

此工作流程自动化了对Reqres.in API的一系列HTTP请求，允许用户检索、创建和更新用户数据。

示例：小企业主可以使用此工作流从Reqres.in API快速获取用户列表，创建一个具有特定详细信息的新用户，然后更新现有用户的职务，而无需编写任何代码。

## 你能做什么

- T 由手动点击触发，允许用户按需执行工作流

- P 执行GET请求以从Reqres.in API检索用户列表

- S 发送POST请求，以创建具有指定名称和作业的新用户

- S 发送PATCH请求以更新现有用户的职位
