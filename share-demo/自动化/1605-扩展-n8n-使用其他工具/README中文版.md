该工作流程是一个Telegram机器人，允许用户请求几个欧洲首都的天气信息，数据显示在使用R编程语言生成的图像中。

示例：用户可以通过发送“/getweather”命令与机器人进行交互，该命令将触发工作流从OpenWeatherMap API获取天气数据，处理R中的数据，并通过Telegram将生成的图像发送回用户。

## 你能做什么

- H 使用开关节点处理不同的机器人命令（例如“/start”、“/getweather”）

- F 从OpenWeatherMap API获取天气数据，并在R中对其进行处理以生成图像

- P 为API和R脚本失败提供错误处理，并向用户发送适当的错误消息
