# 1416向Telegram发送关于已保存到阅读器的新项目的消息

该工作流程自动从Readwise API获取新文章，将它们写入二进制文件，并为每个新文章发送Telegram消息。

示例：想要了解Readwise阅读活动最新信息的用户可以使用此工作流在Readwise帐户中添加新文章时自动接收Telegram通知。

## 你能做什么

- P 定期检查Readwise API中的新文章，并筛选出已处理的文章

- W 将最新文章数据写入二进制文件以进行持久存储

- S 为每篇新文章发送一条电报消息，包括标题、作者、摘要和网址
